# Majestic Admin Binder v2.3.0

Дата выпуска: 11.05.2026

## Сборка одного EXE для отправки администратору

Запусти в корне проекта:

```bat
build_admin_exe.bat
```

Отправляй только:

```text
ProjectFiles\release_admin\MajesticAdminBinder.exe
```

Не отправляй exe из `bin`, `obj`, `Debug` или `Release` без publish.

## Генератор ключей

```bat
ProjectFiles\Tools\run-keygen.bat
```

## Что нового

- версия и дата выпуска в настройках;
- кнопка проверки обновлений-заглушка;
- иконка в трее из логотипа с перекраской в акцент;
- активация использует `Assets/activation.png`;
- базовые ассеты и данные встроены в exe;
- удалены категории сокращений: бизнесы, маршруты, MCL;
- кастомные сокращения показываются первыми и могут удаляться;
- удалено быстрое действие по умолчанию «Открыть репорты»;
- добавлена настройка клавиши открытия репортов рядом с динамиком;
- поддержаны MOUSE3-MOUSE5 и Numpad+/Numpad-/Numpad*/Numpad/ в горячих клавишах.
